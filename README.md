# 🔧配置开发环境
### ⚙️安装依赖

```
yarn install
```

### 🚗编译并运行

```
yarn electron:serve
```

### 📦️编译并打包

```
yarn electron:build
```

### Lints and fixes files

```
yarn lint
```
# 🦎版本要求

nodejs<17
